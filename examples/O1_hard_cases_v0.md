# O1 Hard Cases v0

This file contains the first annotated hard cases for the next O1 difficulty phase.

These cases are harder than the initial seed examples because they involve one or more of the following:

- distributed relations
- implicit comparison criteria
- threshold dependence
- weak identity continuity
- agency metaphors
- compressed causal narratives

The goal is not to solve all hard cases at once.

The goal is to begin testing whether O1 still produces useful structural decompositions when the input is less obvious and less clean.

---

## Case 1 - Motion

### Standard formulation
The river flows

### Hidden observer assumption
Assumes the river is a stable object with an intrinsic action called flowing, rather than a continuously changing material configuration viewed at a macroscopic scale.

### Structural distortion
A distributed process involving water displacement, channel constraint, and observer-selected continuity is compressed into a single object-like action.

### Decentered reformulation
Water continuously changes position through the river channel, and this ongoing displacement is described at the macroscopic level as river flow.

### Emergent structure
Flow is not a simple intrinsic action of a fixed object, but a structured pattern of material displacement under channel constraints.

### Evaluation
- C1 = 1
- C2 = 1
- C3 = 1
- C4 = 1

### O1_GAIN
4

### Verdict
Accepted

## Case 2 - Time

### Standard formulation
The deadline is approaching

### Hidden observer assumption
Assumes an event in calendrical structure can be described as if it were moving toward the observer in time.

### Structural distortion
A relation between current time and a scheduled future constraint is compressed into motion-like language.

### Decentered reformulation
The current time is getting closer to the scheduled time associated with the deadline.

### Emergent structure
"Approaching" is a metaphorical compression of decreasing temporal distance between present state and a fixed scheduled constraint.

### Evaluation
- C1 = 1
- C2 = 1
- C3 = 1
- C4 = 1

### O1_GAIN
4

### Verdict
Accepted

## Case 3 - Causality

### Standard formulation
The market reacted to the news

### Hidden observer assumption
Assumes a distributed set of price changes and trades can be described as if a single unified agent responded directly to a single cause.

### Structural distortion
Multiple actors, models, delays, expectations, and correlated updates are compressed into one linear narrative event.

### Decentered reformulation
Following the release of the news, market prices and trading behavior changed in ways modeled as responses to new information under current conditions.

### Emergent structure
The apparent reaction is a distributed update across many agents and mechanisms, not a single agent-like causal response.

### Evaluation
- C1 = 1
- C2 = 1
- C3 = 1
- C4 = 1

### O1_GAIN
4

### Verdict
Accepted

## Case 4 - Agency

### Standard formulation
The algorithm wants to minimize error

### Hidden observer assumption
Assumes an optimization procedure can be described using intention language without loss of structural clarity.

### Structural distortion
Objective function, update rules, and training dynamics are compressed into human-like wanting.

### Decentered reformulation
The algorithm is configured to update its parameters in ways that reduce the selected error function under the training procedure.

### Emergent structure
The apparent wanting is an optimization relation between system updates, objective function, and training constraints, not an intention.

### Evaluation
- C1 = 1
- C2 = 1
- C3 = 1
- C4 = 1

### O1_GAIN
4

### Verdict
Accepted

---

## Minimal conclusion

These first hard cases suggest that O1 can still produce useful decompositions when the sentence contains:

- metaphorical motion
- distributed causality
- object-process ambiguity
- agency compression

This does not yet show robustness.

It only shows that the next difficulty phase has begun without immediate collapse.
