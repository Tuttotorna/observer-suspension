# O1 Minimal Axioms

This document defines the minimal axiomatic layer of Protocol O1.

These are not final metaphysical claims.

They are operational principles that constrain how observer suspension is applied.

The purpose of this document is to compress O1 into a smaller, harder core.

Without this layer, O1 risks remaining an interesting descriptive procedure.

With this layer, O1 begins to acquire methodological identity.

---

## Why minimal axioms are needed

The repository already contains:

- a general framework
- Protocol O1
- first cases
- domain-focused case sets
- recurring pattern candidates
- a standard application template
- an initial benchmark

At this stage, the correct next move is not indefinite expansion.

The correct next move is to define the smallest set of principles that all O1 applications must respect.

These principles do not complete the framework.

They only define its minimum structural discipline.

---

## Axiom 1 - No privileged human observer by default

No human observer position, local point of view, present-centered access, or lived phenomenological stance may be treated as ontologically privileged by default.

### Meaning
A local human standpoint may be used descriptively.

It may not be silently elevated into the primary center of structure.

### Function
This is the foundational axiom of O1.

Without it, observer suspension does not begin.

---

## Axiom 2 - Apparent absolutes must be tested for dependence

Any apparently absolute property must first be tested for dependence on:

- frame
- relation
- configuration
- access condition
- reversibility structure
- descriptive index

### Meaning
Statements such as:
- the object is still
- the past is fixed
- the sun rises

must not be accepted as structurally primitive without first testing whether they are observer-dependent compressions.

### Function
This axiom drives the recurrent shift:

`absolute -> relational`

---

## Axiom 3 - Phenomenological immediacy is not structural primitivity

A lived experience, no matter how immediate, is not sufficient ground for claiming structural primitivity.

### Meaning
Experiences such as:
- succession
- flow
- presentness
- passive seeing

may be psychologically basic while still failing to be structurally basic.

### Function
This axiom drives the recurrent shift:

`phenomenology -> structure`

---

## Axiom 4 - Readable explanation is not complete structure

A readable directional explanation must not be treated as equivalent to full dependency structure.

### Meaning
Statements such as:
- A causes B
- the event happened because of X
- one decision changed everything

may provide narrative compression without providing structural completeness.

### Function
This axiom drives the recurrent shift:

`narrative arrow -> conditional dependency`

and more generally:

`readability -> compression`

---

## Axiom 5 - Local access indices are not universal centers

Any local access marker such as:

- now
- here
- this point of view
- current state
- present observer position

must be treated first as an index, not as a universal structural center.

### Meaning
What is locally accessible may be important.

But local accessibility does not automatically define ontological privilege.

### Function
This axiom drives the recurrent shift:

`single center -> local index`

---

## Axiom 6 - Container language must be tested against transition structure

Whenever a process is described through container-like language, the description must be tested against state-transition, ordering, or configuration-based reformulation.

### Meaning
Expressions such as:
- in time
- over time
- time passes
- things unfold in time

may hide structural organization behind vague backdrop metaphors.

### Function
This axiom drives the recurrent shift:

`container language -> transition language`

---

## Axiom 7 - O1 reformulation must produce structural gain or fail

A valid O1 application must reveal at least one of the following:

- relational structure
- dependency structure
- indexing structure
- ordering structure
- configuration structure
- access asymmetry
- transition structure

If none of these appear, the reformulation must be treated as weak or trivial.

### Meaning
O1 is not justified by paraphrase alone.

It is justified only by non-trivial structural gain.

### Function
This axiom prevents inflation and protects the method from becoming stylistic rewriting.

---

## Axiom 8 - Observer suspension is methodological, not eliminative

O1 does not eliminate observation.

It suspends only the silent privilege of the human observer as default center.

### Meaning
The framework does not claim:

- no observation
- no frame
- no access
- no perspective

It claims only:

- no privileged human observer by default

### Function
This axiom protects O1 from being misread as anti-observational or pseudo-mystical.

---

## Axiom 9 - Structural recurrence matters more than isolated brilliance

An O1 result gains strength only when the same structural shift recurs across multiple cases or domains.

### Meaning
A single elegant reformulation is not enough.

What matters is recurrence, reuse, and comparability.

### Function
This axiom justifies:

- pattern extraction
- benchmark construction
- future formalization

## Minimal conclusion

These axioms do not complete Observer Suspension.

They only define the minimum discipline required for O1 to function as a method rather than as a loose conceptual intuition.

If future work consistently respects these axioms and continues to generate recurring structural gains, then a stronger formal layer may become justified.

If not, O1 should remain a limited exploratory method.
